# NUMSMU Labs Website

Production-ready Next.js 15+ website for NUMSMU Labs, India's innovation-to-industry ecosystem.

## Stack
- Next.js App Router with React Server Components
- TypeScript
- Tailwind CSS
- Framer Motion
- Shadcn-style component primitives
- Lucide Icons

## Run locally
```bash
npm install
npm run dev
```

## Production
```bash
npm run build
npm run start
```

## Deploy on Vercel
1. Import this repository into Vercel.
2. Use the default Next.js framework preset.
3. Set the production domain to `numsmu.com` or your chosen domain.
4. Deploy. Sitemap and robots are generated automatically.

## Pages
- `/` landing page with hero, animated network, India innovation map, divisions, platform, products, impact and founder sections.
- `/co-founder` recruitment page with roles and application form.
- `/msme-services` detailed MSME offerings.
- `/innovation-platform` AI education platform.
- `/product-development` product studio lifecycle.
- `/resources` resource listing with search/filter UI.
- `/contact` segmented contact form.
